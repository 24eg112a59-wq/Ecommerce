# ShopEZ Backend - Phase 1

This backend is the foundation for the ShopEZ MERN project.

## Included in Phase 1

- Express server setup
- MongoDB Atlas connection via Mongoose
- Middleware setup for JSON, URL-encoded data, and CORS
- Health check route for quick validation
- Environment variable template

## Start Here

1. Copy `backend/.env.example` to `backend/.env`
2. Add your MongoDB Atlas connection string
3. Run `npm install`
4. Start the server with `npm run dev`
